# Wireshark Network Traffic Analysis

## Project Overview
This project demonstrates network packet analysis using Wireshark.
Different types of network packets such as HTTP, DNS, TCP and ICMP
were analyzed to understand network communication.

## Tools Used
Wireshark
Kali Linux

## Tasks Performed

### HTTP Packet Analysis
Captured HTTP packets and extracted User-Agent information.

### DNS Traffic Analysis
Filtered DNS packets and identified DNS transaction IDs.

### TCP Three Way Handshake
Analyzed SYN, SYN-ACK and ACK packets to understand TCP connection process.

### ICMP Packet Analysis
Captured ICMP packets and identified source IP and MAC addresses.
